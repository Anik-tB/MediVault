# MediVault

MediVault is split into three parts:

- `user-mobile`: mobile app for users
- `admin-web`: web dashboard for admin
- `backend`: shared API and database logic

## Current structure

- `user-mobile/`
- `admin-web/`
- `backend/`
- `docs/`

## Suggested build order

1. Build the mobile user app screens and navigation.
2. Build the admin web dashboard.
3. Connect both apps to one backend and database.
